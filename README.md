### 🎮 Gamified Productivity System

- Level progression system  
- Reward unlock mechanics

## ⚙️ Setup Guide

### 1. Clone Repo
```bash
git clone https://github.com/sisodiyaashraf/habito_ai.git
```

### 2. Setup Environment
Create `.env` file:
```env
GEMINI_API_KEY=YOUR_KEY
```

### 3. Install Dependencies
```bash
flutter pub get
```

### 4. Run App
```bash
flutter run
```
